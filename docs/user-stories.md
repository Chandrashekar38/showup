# User Stories

## Commitment Creation

As a user,

I want to create a commitment,

so that I can document my journey.

---

## Progress Updates

As a user,

I want to post updates,

so that I can record my progress.

---

## Following

As a user,

I want to follow commitments,

so that I can learn from others.

---

## Profile

As a user,

I want to view my commitment history,

so that I can reflect on my growth.
