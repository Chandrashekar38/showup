# Product Requirements Document v1

## Product Name

ShowUp

## Problem

People make commitments but rarely preserve the journey behind them.

## Solution

A platform where users:

- Create commitments
- Post progress updates
- Build permanent timelines
- Follow other journeys

## MVP Features

### Create Commitment

Users can create commitments.

### Progress Updates

Users can document progress with text and images.

### Commitment Timeline

Every commitment has its own page.

### User Profile

Shows commitments and updates.

### Follow Commitment

Users can follow journeys.

## Out of Scope

- Money staking
- Charity
- Verification
- Chat
- AI features
- Reputation scores
